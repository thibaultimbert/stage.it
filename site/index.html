Built for the real store workflow

The plugin starts from Shopify product pixels, not a text-only product description, so packaging, proportions, and labels stay anchored to the source listing.

  1. Authenticate with Shopify CLI.
  2. Pull catalog media and write manifests.
  3. Generate 3:4 previews from the product image.
  4. Upload only after explicit approval.
